# Joint-Confident model
Tensorflow implementation of Joint-Confident model proposed by Kimin Lee, in the paper 
[Training Confidence-Calibrated Classifier for Detecting Out-of-Distribution Samples](https://arxiv.org/abs/1711.09325v3).
